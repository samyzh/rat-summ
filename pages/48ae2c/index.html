ES6-ES12的开发技巧

# 总括

# ES6

# 1、let 和 const

这两个的出现，总感觉是为了开发的代码规范而出现的。我们要逐渐放弃var，在项目中多用let和const

与var的区别：

  • var有变量提升，有初始化提升，值可变
  • let有变量提升，没有初始化提升，值可变
  • const有变量提升，没有初始化提升，值不可变，但如果是定义对象，则属性可变
